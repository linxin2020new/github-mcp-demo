# GitHub MCP 演示仓库

这个仓库用于演示 GitHub MCP (Model Context Protocol) 的各种功能。

## 功能演示

1. 创建仓库
2. 添加文件
3. 创建分支
4. 创建拉取请求
5. 管理问题

## 什么是 MCP？

MCP (Model Context Protocol) 是一个开放协议，允许 AI 模型安全地连接到外部数据源和工具。

## GitHub MCP

GitHub MCP 提供了一组工具，可以直接与 GitHub 仓库进行交互，包括：
- 搜索代码
- 管理问题和拉取请求
- 创建和更新文件
- 管理分支和标签
- 等等...

这个仓库就是使用 GitHub MCP 创建的！
